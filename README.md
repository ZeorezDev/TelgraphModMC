# Historical Telegraph

_A lightweight telegraph communication mod for Minecraft Forge 1.20.1_

## Overview

**Historical Telegraph** is a lightweight Minecraft Forge mod for **1.20.1** that adds a simple telegraph-style communication system inspired by early wired communication technologies.

Players can place **Telegraph Machines**, connect them using cables, and create communication networks that transmit messages between locations. The mod is designed to stay simple, immersive, and as close as possible to Minecraft’s vanilla style.

Its main target audience is players who enjoy **historical themes**, **immersive infrastructure**, and **roleplay-oriented gameplay**.

## Features

- Telegraph machines connected through cables
- Network-based message transmission
- GUI-based communication system
- Audio and particle feedback when sending and receiving messages
- Simple cooldown system to prevent spam
- Designed for multiplayer, roleplay, and historical-themed gameplay

## Design Goals

This mod is built around a few simple principles:

- Keep the gameplay lightweight and easy to understand
- Preserve Minecraft’s core style instead of replacing it
- Add historical telegraph-inspired infrastructure in a vanilla-friendly way
- Support roleplay, nations, military, and settlement-based multiplayer servers
- Appeal especially to players interested in historical and semi-historical Minecraft experiences

## Development Note

This project is currently developed by **one person**.

Because of that, newly added blocks currently use **vanilla Minecraft models and existing in-game visual assets** where possible. Custom block models are not planned for now, mainly because there is no dedicated modeler working on the project at this time.

For now, the main focus is on:
- gameplay features
- stability
- simplicity
- historical atmosphere

- **Minecraft Forge 1.20.1**

## Notes

Historical Telegraph is meant to remain a focused communication mod, not a large-scale automation or technology overhaul. The priority is to expand the telegraph system without breaking the game’s basic feel.
